# 📝 Daily Legend Journal

A simple browser-based journaling web app built using HTML, CSS, and JavaScript.

This project is designed for daily self-reflection and self-growth in a fun, motivating way.  
Log in each day, write your thoughts, and build consistency over time.

---

## 🚀 Features

- 🔐 Basic Login System
- 🗓 Daily Journal Entries
- 💾 Local Storage Persistence (data saved in browser)
- 🌙 Clean Dark Theme UI
- 📜 View Previous Entries
- 🚪 Logout Option

---

## 🛠 Built With

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Browser LocalStorage API  

---

## 📂 How To Use

1. Download or clone this repository
2. Open the `index.html` file in your browser
3. Login using the default credentials:
   - Username: `admin`
   - Password: `1234`
4. Start writing your daily entries

---

## 🔒 Privacy

All entries are stored locally in your browser using LocalStorage.  
No external database or server is used.
